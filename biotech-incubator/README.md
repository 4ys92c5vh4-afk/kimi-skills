## 核心能力

### 入孵评审（项目筛选）
- 入孵材料完整性预审 + 质量深度检查
- 项目评审打分（6维度，支持简洁版/详细版）
- 团队配置评估 + 背景核查（红旗信号识别）
- 知识产权评估
- 环评合规检查
- 竞争格局分析
- 技术成熟度评估（TRL）

### 专项评审（按领域）
- 药品注册路径（IND/NDA/ANDA/BLA，NMPA/FDA/EMA）
- 医疗器械注册（II/III类、创新器械、510k/PMA/CE）
- 诊断技术（IVD）专项
- AI+医疗监管（算法更新控制、可解释性）
- 疾病领域评估（肿瘤/心血管/神经/罕见病/抗感染/自免）
- GMP/GLP 审计清单
- 实验室数据合规（ALCOA+）

### 前沿技术评审
- 细胞与基因治疗（CGT）
- 脑机接口（BCI）
- 类器官与再生医学
- 多组学大数据驱动新药研发
- 基因编辑（CRISPR/Base/Prime Editing）
- mRNA技术
- 空间组学
- 微生物组
- 器官芯片
- 合成生物学
- 颠覆性技术识别框架

### 运营辅助
- 批量评审 + 项目排序（P0/P1/P2/P3）
- 生成补充材料通知
- 评审会准备材料（打分表、一页纸摘要）
- 园区/载体对比表
- 入孵后里程碑跟踪 + 退出机制
- 政策匹配建议
- 投资条款与孵化协议建议
- 跨境合作（License-in/out、中美双报、HGRAC）
- 产业方向趋势判断
